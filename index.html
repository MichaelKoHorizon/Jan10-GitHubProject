GitHub Project: What I've Learned in Web Design:

What I've Learned (Tags/Structure):

I learned about many tags such as <h1> as well as <p>.

I also learned about the structure of a html document.

Lists:

Ordered List:

I learned that an ordered list is <ol> with a closing tag and that inside of the list, there is the tag <li>.

Unordered List:

I learned that an unordered list is <ul> with a closing tag and that inside of the list, there is the tag <li>.

Description List:

I learned that an description list is <dl> with a closing tag and that inside of the list, there is the tag <dt> as well as the tag <dd>.

Tables:

I learned that to create a table, you have to start with <table>. After that, you put <tr> which represents the rows. Inside the first row, you can put <th> and afterwards, you can put <td>.

What I learned(Styling):

I learned about three different types of styling: Inline, Internal, External.

Inline:

Inline styling is when you directly style within a tag. For example, <p style="color: blue;">.

Internal styling is when you put all of the styling inside of the head tag on the html document. Inside of the <head> tag, there is going to be a <style> and this is where the styling is going to be.

External styling is when you put all of the styling inside of a seperate .css file. Styling willnot be on the .html document.
